# Handwritten Digit Classification

This is a simple project where I built a neural network to recognize handwritten digits (0–9).

I used TensorFlow and Keras with the MNIST dataset.

## What this project does

* Takes images of handwritten digits
* Trains a neural network model
* Predicts which digit it is

## Tools used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

## How to run

1. Download or clone this repository
2. Open the notebook file
3. Run all cells

## Notes

* The model is simple and beginner-friendly
* Accuracy can be improved using more advanced models like CNN
